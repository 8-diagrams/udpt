UDPT¶

UDPT is a lightweight UDP torrent tracker written in C++, it implements BEP15 of the BitTorrent protocol. This project was developed with simplicity and security in mind. Unlike most tracker, UDPT will save you bandwidth from all that TCP overhead.

Features¶

  • UDP protocol based tracker
  • Simple INI configuration file
  • HTTP REST API
  • SQLite3 database, support for in-:memory:.
  • Logging
  • Linux daemon / Windows Service
  • Choice of static or dynamic tracker modes
